# 📊 Smart Stock Decision Dashboard

## 🔰 Introduction

The Smart Stock Decision Dashboard is a web application designed to help users quickly understand stock market performance and make faster decisions.
It provides real-time stock data along with intelligent insights such as stock trends, top movers, and a personalized watchlist.

---

## 🎯 Objective

The objective of this project is to demonstrate:

* JavaScript fundamentals
* API integration using `fetch`
* Use of array higher-order functions (`map`, `filter`, `sort`)
* UI/UX design skills

---

## 🌐 API Used

This project uses the **Alpha Vantage API** to fetch real-time stock data.

### Endpoints:

* Stock Search
* Stock Price (Global Quote)

---

## ✨ Features

### 🔍 Search Stocks

* Search stocks by name or symbol
* Displays matching results dynamically

### 📊 Stock Insights

* Current stock price
* Percentage change
* Trend indicator:

  * 🟢 Bullish
  * 🔴 Bearish
  * 🟡 Stable

### 🎛️ Filtering

* Filter stocks based on:

  * Price range
  * Performance (gain/loss)

### 🔃 Sorting

* Sort stocks by:

  * Price
  * Change percentage

### ⭐ Watchlist (Key Feature)

* Add/remove stocks
* Stored in localStorage
* Tracks selected stocks

### 📈 Top Movers (Advanced Feature)

* Displays:

  * Top gainers
  * Top losers

---

## 🧠 Use of Higher-Order Functions

This project uses JavaScript array methods extensively:

* `map()` → To render stock data dynamically
* `filter()` → To filter stocks based on conditions
* `sort()` → To rank stocks by price or performance

These improve code readability and efficiency.
